# 🚗 Car Damage Detection using Optimized VGG16 CNN

An AI-powered desktop application that automatically detects vehicle damage from images using an optimized VGG16 Convolutional Neural Network (CNN). This project was developed as my Final Year Project for the Bachelor of Computer Science (Software Engineering) at Universiti Malaysia Pahang Al-Sultan Abdullah (UMPSA).

## 📖 Project Overview

Vehicle damage inspection is commonly performed manually, making it time-consuming and subjective. This project applies deep learning and computer vision techniques to automatically classify vehicle damage into four categories:

- Crack
- Dent
- Scratch
- No Damage

The system combines an optimized VGG16 CNN model with a desktop GUI to provide real-time damage prediction, confidence scores, and research evaluation metrics.
