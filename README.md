# Car Damage Detection using Optimized VGG16 CNN
Machine learning and computer vision project for car damage detection using VGG16 CNN.

🚧 Ongoing Final Year Machine Learning Project

This project focuses on car damage detection using an optimized VGG16 Convolutional Neural Network (CNN) with TensorFlow and computer vision techniques.

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- Scikit-learn
- Matplotlib
- Anaconda Environment

## Project Status
This research project is currently under active development and optimization.
